# Cartella immagini `Unga`

Questa cartella contiene le immagini utilizzate nel portfolio (`index.html`).

## Catalogo file
- `AcconciaturaVogueSposa.JPG` — dettaglio acconciatura/editoriale Vogue Sposa.
- `BraccialeStrass.JPG` — bracciale gioiello con strass.
- `CerchiettoOro.JPG` — cerchietto dorato dal design pulito.
- `DolceEGabbanaAconiature.JPG` — acconciatura couture progetto Dolce & Gabbana.
- `FioriStoffa.JPG` — composizione di fiori in tessuto applicati a mano.
- `GabbiaVogueSposa.JPG` — headpiece a gabbia, progetto Vogue Sposa.
- `GioielliPerle.JPG` — set gioielli con perle.
- `GioielliPerleSwraovski.JPG` — gioielli con perle e cristalli Swarovski.
- `SfilataDolceGabbana.JPG` — look/accessori da sfilata Dolce & Gabbana.
- `SpillaGioiello.JPG` — spilla decorativa con perle e cristalli.
- `TiaraGioiello.JPG` — tiara gioiello per styling editoriale.
- `VogueSposa.JPG` — immagine editoriale Vogue Sposa.
- `VogueSposaAfrica.JPG` — variante editoriale Vogue Sposa (Africa).
- `VogueSposaIkebana.JPG` — variante editoriale Vogue Sposa (Ikebana).
- `cerchiettoGioiello.JPG` — cerchietto gioiello (versione dettagliata).
- `gioielliZar.JPG` — composizione gioielli con dettagli metallici.

## Note
- Mantenere i nomi file coerenti con i riferimenti in `index.html`.
- Evitare rinomine senza aggiornare i percorsi nel portfolio.
